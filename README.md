# Simulador Exchange de Cripto Monedas

La aplicación permite comprar, vender y cambiar criptomonedas. 
Tambien crear perfil y loguearte con tu cuenta.

# Tecnologías: 
    - Java
    - Spring Boot
    - Spring Security / JWT
    - MySQL
    - JPA   
    - HTML
    - CSS
    - Bootstrap
    - Angular
